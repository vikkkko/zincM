# Comparison operators

### Equality

`==` is a binary operator.

*Accepts*
1. Integer **or** boolean expression
2. Expression of the operand 1 type

*Returns* the boolean result.

### Non-equality

`!=` is a binary operator.

*Accepts*
1. Integer **or** boolean expression
2. Expression of the operand 1 type

*Returns* the boolean result.

### Lesser or equals

`<=` is a binary operator.

*Accepts*
1. Integer expression
2. Expression of the operand 1 type

*Returns* the boolean result.

### Greater or equals

`>=` is a binary operator.

*Accepts*
1. Integer expression
2. Expression of the operand 1 type

*Returns* the boolean result.
