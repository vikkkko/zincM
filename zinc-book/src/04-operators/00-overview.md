# Precedence

The top one is executed first.

|              Operator              |    Associativity    |
|------------------------------------|---------------------|
| ::                                 | left to right       |
| [] .                               | left to right       |
| - ~ !                              | unary               |
| as                                 | left to right       |
| * / %                              | left to right       |
| + -                                | left to right       |
| << >>                              | left to right       |
| &                                  | left to right       |
| ^                                  | left to right       |
| ⎮                                 | left to right       |
| == != <= >= < >                    | require parentheses |
| &&                                 | left to right       |
| ^^                                 | left to right       |
| ⎮⎮                               | left to right       |
| .. ..=                             | require parentheses |
| = += -= *= /= %= ⎮= ^= &= <<= >>= | require parentheses |
